# A4 Closure Note, Two-Hive Seeds 100..129

This note closes the current A4 two-hive research loop using existing artifacts
only. It records the interpretation boundary after the holdout, delayed-coupling
temporal null, and accounting-control analyzer.

No simulations were rerun for this closure note. No simulator mechanics,
three-hive coupling, new lobe labels, dashboards, LLM calls, Lean, Slack,
browser automation, Atomspace integration, or external services were added.

## Decision

Freeze A4 as a queue-flow/service and action-opportunity accounting result with
a documented delayed synchronization diagnostic. The seed `100..129` artifacts
do not justify an independent multi-hive lobe-grammar claim, strange-attractor
claim, or immediate three-hive implementation.

The A4 artifact contract and read-only analyzers are stable enough for bounded
scientific comparison:

- full-transfer coupling is implemented, conserved, and analyzable;
- direct and shuffled are equivalent in the two-hive target-assignment setting,
  so `direct minus shuffled` is a schema/source-opportunity contract check, not
  a phase-randomization null;
- delayed coupling produces a raw completion-fraction synchrony diagnostic
  that survives the circular-shift temporal null;
- the same delayed completion-fraction synchrony no longer remains outside the
  residualized circular-shift null after non-tautological opportunity/load and
  action-accounting controls.

## Evidence Boundary

The holdout summary in `docs/results/a4_two_hive_holdout_seed100_129.md`
reported bootstrap-supported delayed-minus-none effects on completion-fraction
correlation at lags `0` and `2`. The follow-up temporal null in
`docs/results/a4_delayed_coupling_null_seed100_129.md` kept those two
completion-fraction endpoints outside the circular-shift null while leaving
load-backlog correlation, queued-age divergence, and completion-fraction
divergence inside the conservative interpretation band.

The accounting-control analyzer in
`docs/results/a4_accounting_control_seed100_129.md` then residualized
completion-fraction trajectories and reran the same circular-shift null. The
primary combined non-tautological controls included clock trend, queue/load,
queued-age, work/action mix, and transfer-timing covariates. Under those
controls, delayed-minus-none completion-fraction correlation at both lag `0`
and lag `2` fell inside the residualized null interval.

That pattern supports the conservative interpretation: delayed full-transfer
coupling changes queue-flow timing, service opportunity, load state, and action
mix in a way that can produce completion-fraction synchrony. It does not show a
residual coordination mechanism independent of those accounting fields.

## Stop Conditions

Do not reopen the current A4 loop for more simulator mechanics, three-hive
coupling, new lobe architectures, broad residual-lobe sweeps, or external
integrations from the seed `100..129` result alone.

The only acceptable reasons to reopen A4 are:

- a concrete artifact or analyzer bug that changes the recorded results;
- a narrowly scoped read-only validation over existing artifacts that does not
  promote synchronization diagnostics to mechanism claims;
- a separate preregistered future design requested explicitly before any new
  simulator mechanics are added.

## Future Boundary

If future work asks for a three-hive or richer multi-hive design, the next
artifact should be a preregistration, not simulator code. That design must
define target-null semantics, transfer-volume matching, source-opportunity
matching, delivery-timing controls, seed streams, conservation tests,
paired-seed uncertainty, and decision rules before implementation.
