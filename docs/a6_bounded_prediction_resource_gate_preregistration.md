## Scientific Question

Does explicit bounded prediction-resource accounting create an intermediate
budget regime that is more residually predictable or compressible than zero
budget and high/oracle prediction, while preserving matched work opportunity
and beating linear, shuffled, and replay controls?

This gate tests a resource mechanism, not a lobe or attractor claim.

## Candidate Rule

The candidate condition passes only if the same intermediate-budget condition:

```text
beats zero budget on residual recurrence or compression
beats high/oracle smoothing on budget efficiency
beats amplitude-matched linear prediction
beats phase-shuffled delayed signal
beats threshold-shuffled thresholds
beats budget-matched prediction replay
has acceptable confound_r2 after accounting removal
preserves or improves artifact maturity, provenance debt, and risk
```

If any matched control or replay null matches the candidate on the primary
residual endpoint, the gate fails closed.
